# Boss直聘自动投递脚本

基于Selenium的Boss直聘简历自动投递工具

## 功能特点

- 自动搜索职位并投递简历
- 自动发送招呼语
- 防重复投递（每日记录）
- 随机延时模拟真人操作
- 支持自定义搜索条件和投递配置

## 环境要求

- Python 3.7+
- Chrome浏览器
- ChromeDriver（需与Chrome版本匹配）

## 安装依赖

```bash
pip install selenium
```

## 使用步骤

### 1. 配置文件

编辑 `config.json` 设置搜索条件：

```json
{
  "搜索配置": {
    "关键词": "Python开发",
    "城市": "北京",
    "薪资范围": "15-30K",
    "工作经验": "3-5年"
  },
  "投递配置": {
    "每日上限": 50,
    "招呼语": "您好，看到贵公司的招聘信息...",
    "最小间隔秒数": 3,
    "最大间隔秒数": 8
  }
}
```

### 2. 运行脚本

```bash
cd boss_auto
python main.py
```

### 3. 扫码登录

程序启动后会打开浏览器，请在浏览器中扫码登录Boss直聘

### 4. 自动投递

登录成功后会自动开始搜索和投递

## 文件说明

- `main.py` - 主程序
- `config.json` - 配置文件
- `delivered.json` - 已投递记录（自动生成）

## 注意事项

1. 首次运行需要手动扫码登录
2. 投递记录按日期保存，每天会重置
3. 建议设置合理的每日上限，避免频繁操作被限制
4. 招呼语可以自定义，支持模板变量
5. ChromeDriver需要与Chrome浏览器版本匹配

## 常见问题

**Q: 提示找不到ChromeDriver？**

A: 需要下载与Chrome浏览器版本匹配的ChromeDriver，并添加到系统PATH或放在脚本目录

**Q: 投递失败或元素找不到？**

A: Boss直聘页面可能更新，需要根据实际页面调整CSS选择器

**Q: 如何查看已投递记录？**

A: 查看 `delivered.json` 文件
